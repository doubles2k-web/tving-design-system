Usage Guidance
✓ Do
tving.red는 CTA 버튼, 브랜드 강조에만 단독 사용. Opacity 변형(red·15, red·10)으로 배경 표현 시 사용합니다.
✗ Don't
tving.red를 일반 텍스트 색상으로 사용하지 않습니다. 흰 배경 위에서 red·50 이하 Opacity는 WCAG 기준 미달이므로 단독 텍스트 사용 금지.

Usage Guidance
✓ Do
Bold는 1~2줄 헤드라인에만. Read는 3줄 이상 본문 텍스트에 사용. line-height 차이를 반드시 지킬 것.
✗ Don't
Bold 스타일을 긴 본문에 사용하거나, Read 스타일의 line-height를 임의로 줄이지 않습니다. 최소 폰트 사이즈 10px 미만 사용 금지.
