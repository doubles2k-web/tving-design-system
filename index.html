Color Stable
tving.red를 중심으로 한 단일 브랜드 컬러 전략. Opacity 변형을 통해 다양한 UI 상태를 표현합니다.
Brand Color · Tint
tving.red (#FF153C)가 유일한 브랜드 컬러입니다. 나머지는 콘텐츠 카테고리 구분 및 서비스별 구분에만 사용합니다.
Gray Color · 다이나믹/제플린 공용 토큰
NONE 주의: 아이콘은 원칙적으로 화이트(투명)만 사용합니다. gray500이상만 서비스에 사용 가능합니다. 그 외 4.5이하 비율의 색상은 사용 금지입니다.

ℹ

Overview
TDS › Components › Button
Button Stable
사용자 액션을 트리거하는 기본 인터랙션 컴포넌트. Primary, Secondary, Ghost 3가지 변형을 제공합니다.
Variants
3가지 기본 버튼 변형. 페이지 내 Primary는 1개만 사용합니다.
Size
Usage Guidance
✓ Do
Primary 버튼은 CTA(Call To Action) 1개에만 사용. 배경색은 tving.red(#FF153C) 고정. 텍스트는 Bold, 14px 이상 유지.
✗ Don't
동일한 화면에 Primary 버튼을 2개 이상 배치하지 않습니다. 버튼 레이블을 아이콘만으로 구성하지 않습니다(접근성 위반).
